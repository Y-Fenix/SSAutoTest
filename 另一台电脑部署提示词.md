# SSAutoTest 另一台电脑一键部署提示词

把下面这段话发给另一台电脑上的 Codex：

```text
请帮我在这台 macOS 电脑上一键部署 SSAutoTest 局域网常驻版本。

我已经把部署包解压到本机，请你执行以下事项：

1. 进入解压后的 `SSAutoTest` 项目目录。
2. 检查是否安装 Node.js 20+ 和 npm；如果没有，请告诉我需要先安装 Node.js。
3. 执行 `npm ci` 安装依赖。
4. 执行 `npm test` 和 `npm run build` 验证项目可用。
5. 执行 `printf '\n' | ./一键启动局域网版本.command` 安装并启动局域网常驻服务。
6. 告诉我局域网访问地址、日志目录，以及如何停止服务。

注意：
- 这是 macOS 用户级常驻服务，登录当前 macOS 用户后会自动启动。
- 关闭终端后服务仍会继续运行。
- 如果需要读取飞书表格，请确认这台电脑已经配置并登录 `lark-cli`。
- 如果 GitHub 可用，也可以直接从 `https://github.com/Y-Fenix/SSAutoTest` 拉取最新版。
```

手动部署方式：

```bash
cd SSAutoTest
npm ci
npm test
npm run build
printf '\n' | ./一键启动局域网版本.command
```

停止服务：

```bash
./停止局域网常驻服务.command
```
